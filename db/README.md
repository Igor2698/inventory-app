# Database Schema

This folder contains the SQL schema for MySQL Workbench.

## Files

- `schema.sql` — relational schema for `orders` and `products` tables with foreign keys.

## How to open in MySQL Workbench

1. Open MySQL Workbench.
2. Go to `File` -> `Open SQL Script...`.
3. Select `db/schema.sql`.
4. Execute the script to create the schema in your MySQL instance.

Optional: after importing, use Workbench modeling tools to generate an EER diagram and save it as an `.mwb` file.
